# SIMI API

API intermedia para conectar la web comercial de SIMI con Supabase y futuras integraciones.

## Endpoints

- `GET /health`: verifica que la API este activa.
- `POST /api/leads`: registra un lead desde la landing comercial.

## Variables

Usa `.env.example` como base. Las claves reales deben ir en `.env` o en Vercel, nunca en Git.
